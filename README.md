# kenya-an-gambiae-16S-project
Metadata associated with 16S rRNA gene sequencing of Kenyan Anopheles gambiae s.s.<br>
These were 2-3d non-blood-fed, adult, female, F1 progeny, derived from Bungoma County between April-May 2018.

## Citation<br>
